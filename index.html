리듬 캐피탈 리메이크법

제작 준비

  1. 작품 좋아요와 북마크, 계정 팔로우를 한다
  2. 작품을 리메이크하여 만들기 탭으로 작품을 본다
  3. game이라는 장면으로 이동한다

기초

곡 삭제하는법

  1. 곡의 비트맵과 정보를 삭제한다
  2. 배경 오브젝트에서 삭제하고픈 사운드를 삭제한다
  3. 속성에서 playlist 리스트의 삭제했던 사운드와 같은 번호였던 항목을 삭제한다
  4. 속성에서 곡 선택 변수의 기본값을 고치고 싶다면 고친다

곡 추가하는법

  1. 배경 오브젝트에서 추가하고픈 사운드를 추가한다
  2. 속성에서 playlist 리스트에 방금 추가한 사운드와 같은 번호가 되는 항목 번째에 새 항목을 넣고 항목값으로 그 사운드의 제목을 입력한다
  3. 속성에서 곡 선택 변수의 기본값을 고치고 싶다면 고친다
  4. 곡의 비트맵과 정보를 설정하고 싶으면 설정한다

비트맵 삭제하는법

  1. 비트맵의 정보를 삭제한다
  2. "노트 생성" 함수로 들어간다
  3. 당신이 삭제하고 싶은 비트맵과 실행되는 조건이 같은 실행되는 코드를 찾는다
  4. 그런 코드중 노트를 생성하는 코드를 모두 지운다

비트맵 만드는법

  1. "노트 생성" 함수로 들어간다
  2. 당신이 만들고싶은 비트맵과 실행될 조건이 같은 코드 위치를 찾는다 없으면 직접 만든다
  3. 방금 찾은 위치에 "(노트 생성)~"이라는 긴 함수가 있는데 이걸 조합한걸 넣어 노트를 만든다
  4. 비트맵의 정보를 설정한다

정보 삭제법

  1. "정보 설정"함수로 들어간다
  2. 당신이 삭제하고 싶은 정보가 설정되는 조건과 같은 조건일때 실행되는 코드를 찾는다
  3. 그런 코드중 삭제하고 싶은 정보를 설정하는 코드를 삭제한다

정보 설정법

  1. "정보 설정"함수로 들어간다
  2. 당신이 설정하고 싶은 정보가 설정되는 조건과 같은 조건일때 실행되는 코드 위치를 찾는다 없으면 직접 만든다
  3. 방금 찾은 위치에 "정보"리스트를 넣어 정보를 설정한다

정보 리스트-의미, 사용법

  1. 의미:첫번째 항목은 곡의 길이를 나타내는 항목이다
    사용법:설정하려는 곡의 길이가 "몇초인지" 입력한다
  2. 의미:두번째 항목은 해당 비트맵의 숏노트 개수를 나타내는 항목이다
    사용법:이 코드와 실행되는 조건이 같은 비트맵의 숏노트 개수를 입력한다
  3. 의미:세번째 항목은 해당 비트맵의 롱노트 개수를 나타내는 항목이다
    사용법:이 코드와 실행되는 조건이 같은 비트맵의 롱노트 개수를 입력한다
  4. 의미:네번째 항목은 해당 곡에서 지원하는 종목을 정하는 항목이다(comment)
    사용법:곡에서 지원하려는 종목을 나타내는 번호를 모두 입력하면 된다(예: 12,03,0123)
  5. 의미:다섯번째 항목은 해당 곡에서 지원하는 레벨의 개수를 정하는 항목이다
    사용법:곡에서 지원하려는 레벨의 개수를 입력하면 된다
  6. 의미:여섯번째 항목은 해당 곡에서 지원하는 배경의 모양을 정하는 항목이다
    사용법:곡에서 지원하려는 배경의 모양의 번째를 입력하면 된다
  7. 의미:일곱번째 항목은 해당 곡에서 지원하는 시작버튼의 모양을 정하는 항목이다
    사용법:곡에서 지원하려는 시작버튼의 모양의 번째를 입력하면 된다
  8. 의미:여덟번째 항목은 해당 곡을 실행시킬수 있는지를 정하는 항목이다
    사용법:실행시킬수 있게 하려면 1이란 값을 아니면 0이란 값을 입력하면 된다
  9. 의미:아홉번째 항목은 해당 비트맵에서 지원하는 키 개수를 정하는 항목이다
    사용법:지원하려는 키 개수를 입력하면 된다 (비트맵중 4키가 아닌것이 있다면 직접 호환시켜주는 코드를 만들어야 한다 이 방법은 아래 서술하겠다)
  10. 의미:열번째 항목은 해당 곡을 작곡한 작곡가를 나타내는 항목이다
    사용법:해당 곡을 작곡한 작곡가의 이름을 입력하면 된다
  11. 의미:열한번째 항목은 해당 비트맵을 만든 메이커를 나타내는 항목이다
    사용법:해당 비트맵을 제작한 메이커의 이름을 입력하면 된다

심화

원하는 키의 비트맵 호환시키는법(6키,10키 등등..)

  1. "노트 히트" 함수로 들어간다
  2. 쫄지 않는다
  3. 당신이 호환시키고 싶은 비트맵 키 개수일때 코드를 실행시키는 조건문을 찾는다 없으면 만든다
  4. 방금 찾은 코드가 참일때 실행하는 위치에 히트버튼과 상호작용 하는 키를 만드는 블록을 넣어 모든 히트버튼이 자신과 상호작용 하는 키가 생기도록 만들어준다

히트버튼과 상호작용 하는 키를 만드는 블록 제작법

  1. "노트 히트" 함수로 들어간다
  2. 쫄지 않는다
  3. (시간이 없어 서술하지 못했습니다..)

히트버튼과 상호작용 하는 키 바꾸는법

  1. "노트 히트" 함수로 들어간다
  2. 쫄지 않는다
  3. (시간이 없어 서술하지 못했습니다..)
